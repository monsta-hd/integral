  • bessel_j (n; x) // Bessel's function of the first kind, n is integer
  • jO x or j_O x // Bessel's function of the first kind with n = 0
  • bessel_y (n; x) // Bessel's function of the second kind, n is integer,  x > 0
  • yO x or y_O x // Bessel's function of the second kind with n = 0, x > 0
  • Improper integrals (-inf ; +inf) are computed using 1024- and 2048-nodes Gauss-Hermite quadrature.
  • Improper integrals (a ; +inf) and (-inf ; a) are computed using 512- and 1024-nodes Gauss-Laguerre quadrature.
  • Finite domain integrals (a ; b) are computed using G25 K51 Gauss-Kronrod quadrature with interval subdivision. Computation finishes, when absolute error is less than 1.0e-6 or relative error is less than 1.0e-8, or when specified computation time is out.
